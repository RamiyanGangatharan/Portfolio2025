# Portfolio 2025

Welcome to my new and improved developer portfolio.

This project is a full rebuild of my previous site — redesigned and re-engineered to reflect how far I’ve come as a developer. With my graduation from Durham College and a completed internship behind me, this portfolio is now my ongoing side project, running from **May 2025 to September 2025**, and built to evolve with me over time.

## 🔍 Goals

- Build a responsive, modern single-page application (SPA)
- Showcase selected projects with technical breakdowns
- Highlight skills through real-world implementation
- Create a flexible, extensible architecture
- Maintain a **developer blog** to document progress, challenges, and learning

## 🛠️ Tech Stack (Planned/Current)

- **Frontend**: React + TypeScript + Tailwind CSS + Framer Motion
- **Routing**: React Router or Next.js (SPA mode)
- **Content**: Markdown/MDX for blog and CMS integration (e.g., Sanity)
- **Backend**: Node.js + Express or Flask, optional serverless API
- **Database**: MongoDB (Atlas) or Supabase (PostgreSQL)
- **Forms**: React Hook Form + Zod
- **Deployment**: Vercel / Netlify for frontend, Docker or VPS for full stack
- **CI/CD**: GitHub Actions

## ✍️ Developer Blog

As part of this project, I'm maintaining a blog to share my journey:  
- Dev logs and weekly updates  
- Deep dives into technical decisions  
- Design ideas and performance optimization  
- Wins, blockers, and how I overcame them

This blog will live as its own section on the portfolio, and serve both as a timeline and a behind-the-scenes look into the work.

As the site grows, I’ll continue to add new features, refine the UX, and adopt emerging tools and techniques that push my skills further.
