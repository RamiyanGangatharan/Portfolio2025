# Portfolio 2025

Welcome to my new and improved developer portfolio.

This project is a complete overhaul of my original portfolio site, built to reflect my growth as a software developer and to serve as a long-term, evolving showcase of my work, skills, and interests.

Having recently graduated from Durham College and completed an internship, I'm now building this as a personal side project to highlight what I’ve learned and what I can do — both technically and creatively.

## 🔍 Goals

- Create a responsive, modern, single-page application (SPA)
- Showcase selected projects, technical skills, and writing
- Build a scalable and maintainable architecture for future additions
- Explore modern technologies and concepts through real implementation
